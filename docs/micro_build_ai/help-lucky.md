# 🐾 Help Lucky get to the park

Lucky wants to play 🎾 — but he's a little lost. Let's help him!

## 1. He needs a sign

Lucky can't read many words, so keep it big and clear:

````
# 🐾 ➡️ 🌳
### 🎾 **PARK!** ☀️🐿️
````
{: .block }
