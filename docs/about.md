# ℹ️ About Lightcodepedia

**Lightcodepedia** is an open-access platform for building digital skills through no-code and low-code tools.

Each instance is a **LightNode** — a self-hosted, forkable GitHub Pages site managed by community educators called **LightCoders**.

[👤 Author](/author) · [🕸️ Network](/nodes)

## 🌍 What it is

- A network of LightNodes, each customizable and deployable in minutes
- A library of interactive components: Python runners, AI agents, datasheets, charts, maps and more
- A teaching framework that scales from classroom to global community
