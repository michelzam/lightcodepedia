### What is LightCode?
A no-code/low-code platform for learning how to build apps with little or no code.

### Who can use it?
Anyone — no programming experience needed. It's designed for learners, educators, and citizen developers.

### How much does it cost?
Free. Hosted on GitHub Pages and Streamlit Community Cloud.

### Can I run my own LightNode?
Yes. Fork the repo and follow the [Deploy guide](../deploy). Every node joins the [Network](../nodes).
